# Servants of Bharat

A web-based volunteer management platform that replaces fragmented WhatsApp-based coordination with a centralized system. Volunteers can participate in social service activities, submit proof of their work, track milestone progress, and earn QR-verifiable digital certificates.

---

## Features

- User Registration & Authentication
- 5-Star Volunteer Milestone Tracker
- Event Notice Board
- Proof Submission & Admin Review
- QR-Verifiable Digital Certificates
- In-App Notifications
- Role-Based Access Control

---

## User Roles

- Volunteer
- Event Manager
- Super Admin
- External Certificate Verifier (Public)

## Branch Strategy

This project is maintained in a single GitHub repository with separate branches for frontend and backend development.

| Branch | Purpose |
|---------|---------|
| `main` | Repository documentation and project overview |
| `dev-frontend` | Frontend development |
| `uat-frontend` | Frontend User Acceptance Testing (UAT) |
| `main-frontend` | Frontend production-ready code |
| `dev-backend` | Backend development |
| `uat-backend` | Backend User Acceptance Testing (UAT) |
| `main-backend` | Backend production-ready code |

> **Note:** Frontend branches contain only the frontend application, while backend branches contain only the backend application.
